# 🚀FaculdadeConectado

Plataforma acadêmica para estudantes do INSTIC

> ⚠️ Cansado de grupos de WhatsApp confusos?
> O FaculdadeConectado organiza tudo em um só lugar: avisos, conteúdos e interação acadêmica.

---

## 🎯 Objetivo

Criar um sistema simples e eficiente para:

* 📡 Compartilhar avisos acadêmicos organizados
* 📚 Disponibilizar materiais de estudo (PDFs, resumos)
* 🧠 Engajar estudantes com quizzes e ranking
* 🤝 Facilitar colaboração entre estudantes

---

## 🧩 Funcionalidades (MVP)

* [ ] Sistema de autenticação (login/registro)
* [ ] Criação e listagem de avisos
* [ ] Upload de conteúdos
* [ ] Sistema de quiz
* [ ] Ranking de usuários

---

## 🛠️ Tecnologias utilizadas

* Backend: Go (Gin)
* Banco de dados: PostgreSQL
* Frontend: HTML, CSS, JavaScript

---

## ⚙️ Como rodar o projeto

### Pré-requisitos

* Go instalado
* PostgreSQL instalado

### Passos

```bash
git clone https://github.com/Simao-Mvuti/FauConectado.git
cd FacuConectado
go mod tidy
go run main.go
```
